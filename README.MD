# AVB Power Tool

## Brief

As is known to all, AOSP has a tool to create Android Verified Boot(AVB) footer, avbtool.py. However, this script is designed to work with build workflow, which is unfriendly to ROMers with requirements of AVB.

This tool is designed to make using avbtool.py easier by providing automatical command generation and config management, its functions:

- Read AVB config from image files in current work directory.
- Sign images in work directory with a config file generated by AVB Power Tool.
- Manage configs for better management.

**IMPORTANT**: You CANNOT generate images with AVB footer without public key files. SHA1 digest of key file from images in your working dir cannot use as a public key file.

## Quick Start

Run these commands in your terminal:

```
git clone https://github.com/WASDDestroy/AVBPowerTool.git
cd AVBPowerTool
python3 ./Core/Frontend/HomePageUI.py # Consider use py or python if you are on Windows
```

Then start using it.
